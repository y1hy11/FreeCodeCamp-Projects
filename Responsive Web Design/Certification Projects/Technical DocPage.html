Introdution

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming.

What you should already know

  • A general understanding of computer programming concepts.
  • Good working knowledge of basic data types and operators.
  • Some programming experience.
  • If you are new to programming, try one of the tutorials linked on the main page about Python.
Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules. A Python identifier must start with a letter or underscore (_); subsequent characters can also be digits (0-9). Because Python is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase). You can use Unicode letters such as å and ü in identifiers. You can also use the Unicode escape sequences as characters in identifiers. Some examples of legal names are number_hits, temp99, and _name.

Declaring variables

Declaring variables in Python is a way to store a value in a named location. Here are some examples:

  • Example 1: Declaring a single variable x = 5 In this example, x is the variable name, and 5 is the value assigned to it. You can then use x in your code to refer to the value 5.

  • Example 2: Declaring multiple variables

    x, y, z = 1, 2, 3 This is equivalent to declaring each variable separately: x = 1
    y = 2
    z = 3

  • Example 3: Declaring variables with different data types

    name = "John" # string
    age = 30 # integer
    height = 1.75 # float
    is_admin = True # boolean
    In this example, we declare four variables with different data types: name is a string, age is an integer, height is a float, and is_admin is a boolean.

  • Example 4: Declaring variables with assignment operators

    x = 5 x += 2 # x is now 7
    x -= 1 # x is now 6
    x *= 2 # x is now 12
    x /= 2 # x is now 6.0     In this example, we use assignment operators to modify the value of x. The += operator adds 2 to x, the -= operator subtracts 1 from x, the *= operator multiplies x by 2, and the /= operator divides x by 2.

  • Example 5: Declaring variables with string formatting

    name = "John"
    age = 30
    greeting = "Hello, my name is {name} and I am {age} years old."
    print(greeting) # Output: Hello, my name is John and I am 30 years old.     In this example, we use string formatting to create a greeting message that includes the values of name and age.

  • Example 6: Declaring variables with list and dictionary

    fruits = ["apple", "banana", "cherry"]
    person = {"name": "John", "age": 30, "city": "New York"}     In this example, we declare two variables: fruits is a list of strings, and person is a dictionary with three key-value pairs.

  • Example 7: Declaring variables with tuple

    coordinates = (40.7128, -74.0060) In this example, we declare a variable coordinates with a tuple value.
These are just a few examples of how you can declare variables in Python. Remember to always use meaningful variable names and follow the PEP 8 style guide for naming conventions.

Data Types in Python

In Python, a data type is a classification of data that determines the type of value a variable can hold, the operations that can be performed on it, and the amount of memory it occupies. Python has several built-in data types, which can be categorized into several groups:

Numeric Types
  • Integers (int): Whole numbers, e.g., 1, 2, 3, etc.
  • Floating Point Numbers (float): Decimal numbers, e.g., 3.14, -0.5, etc.
  • Complex Numbers (complex): Numbers with real and imaginary parts, e.g., 3+4j, -2-3j, etc

    • Example:

    x = 5 # integer y = 3.14 # float z = 3 + 4j # complex
Sequence Types
  • Strings (str): A sequence of characters, e.g., "hello", 'hello', etc. Strings can be enclosed in single quotes or double quotes.
  • Lists (list): A collection of items that can be of any data type, e.g., [1, 2, 3], ["a", "b", "c"], etc.
  • Tuples (tuple): A collection of items that cannot be changed, e.g., (1, 2, 3), ("a", "b", "c"), etc.

    • Example:

    name = "John" # string numbers = [1, 2, 3] # list colors = ("red", "green", "blue") # tuple
Mapping Types
  • Dictionaries (dict): A collection of key-value pairs, e.g., {"name": "John", "age": 30}, etc.

    • Example:

    person = {"name": "John", "age": 30, "city": "New York"}
Set Types
  • Sets (set): A collection of unique items, e.g., {1, 2, 3}, {1, 2, 2, 3}, etc.
  • Frozensets (frozenset): An immutable set, e.g., frozenset({1, 2, 3}), etc.

    • Example:

    numbers = {1, 2, 3} # set frozen_numbers = frozenset({1, 2, 3}) # frozenset
Boolean Type
  • Booleans (bool): Represents the truth values True and False.

    • Example:

    is_admin = True # boolean is_active = False # boolean
None Type
  • None (NoneType): Represents the absence of a value or a null value.

    • Example:

    x = None # None
Special Types
  • Ellipsis (ellipsis): Represents a placeholder for code that has not been implemented yet.
  • NotImplemented (NotImplementedType): Represents a method that has not been implemented yet.
  • ModuleType: Represents a module.
  • FunctionType: Represents a function.
  • MethodType: Represents a method.
  • CodeType: Represents a code object.
  • FrameType: Represents a frame object.
  • TracebackType: Represents a traceback object.
  • GeneratorType: Represents a generator object.
  • CoroutineType: Represents a coroutine object.
  • AsyncGeneratorType: Represents an asynchronous generator object.
  • GetSetDescriptorType: Represents a get-set descriptor object.
  • MemberDescriptorType: Represents a member descriptor object.

Conditional Statements in Python

Conditional statements are used to execute different blocks of code based on certain conditions or decisions. In Python, there are several types of conditional statements:

If Statement

The if statement is used to execute a block of code if a condition is true. It has the following syntax:

if condition: # code block

Example:

x = 5 if x > 0: print("x is positive")

This code will print "x is positive" because the condition x > 0 is true.

If-Else Statement

The if-else statement is used to execute one block of code if a condition is true and another block of code if the condition is false. It has the following syntax:

if condition: # code block 1 else: # code block 2

Example:

x = -5 if x > 0: print("x is positive") else: print("x is negative")

This code will print "x is negative" because the condition x > 0 is false.

If-Elif-Else Statement

The if-elif-else statement is used to execute one block of code if a condition is true, another block of code if another condition is true, and a default block of code if all conditions are false. It has the following syntax:

if condition1: # code block 1 elif condition2: # code block 2 else: # code block 3

Example:

x = 0 if x > 0: print("x is positive") elif x < 0: print("x is negative") else: print("x is zero")

This code will print "x is zero" because the condition x > 0 and x < 0 are false, and the else block is executed.

Nested If Statement

Nested if statements are if statements within if statements. They are used to test multiple conditions. It has the following syntax:

if condition1: if condition2: # code block

Example:

x = 5 y = 10 if x > 0: if y > 0: print("x and y are positive")

This code will print "x and y are positive" because both conditions x > 0 and y > 0 are true.

Conditional Expressions

Conditional expressions are used to assign a value to a variable based on a condition. It has the following syntax:

variable = value1 if condition else value2

Example:

x = 5 result = "positive" if x > 0 else "negative" print(result)

This code will print "positive" because the condition x > 0 is true.

These are the basic conditional statements in Python. Understanding how to use them is essential for writing efficient and effective code.
